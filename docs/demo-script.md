# Demo Script: SwasthyaSaathi Lite

## Setup Before Demo
1. Ensure backend is running on Railway
2. Ensure frontend is deployed on Vercel
3. Have WhatsApp sandbox connected on phone
4. Open web chat in browser as backup

## Demo Flow (3-4 minutes)

### 1. Introduction (30 seconds)
- Show the problem: Rural India lacks healthcare information access
- Introduce SwasthyaSaathi Lite: AI healthcare chatbot via WhatsApp

### 2. Symptom Query (60 seconds)
- **Send on WhatsApp:** "I have fever and headache for 2 days"
- **Show:** AI-generated health guidance in simple language
- **Highlight:** Doctor visit recommendation, medical disclaimer

### 3. Multilingual (45 seconds)
- **Send in Hindi:** "मुझे बुखार है"
- **Show:** Hindi response with same quality guidance
- **Highlight:** Automatic language detection

### 4. Vaccination Info (30 seconds)
- **Send:** "What vaccines does my 6-month baby need?"
- **Show:** Vaccination schedule from government guidelines

### 5. Outbreak Alert (30 seconds)
- **Trigger** mock dengue alert
- **Show:** Alert message received on WhatsApp with precautions

### 6. Web Chat (30 seconds)
- Show the web interface for judges to try
- Demonstrate language switcher

### Key Talking Points
- ✅ Works on WhatsApp (no app download needed)
- ✅ Multilingual (English + Hindi)
- ✅ AI-powered but medically safe (disclaimers everywhere)
- ✅ Free to deploy (Vercel + Railway + Twilio free tiers)
- ✅ Targets 600M+ rural Indians with limited healthcare access
